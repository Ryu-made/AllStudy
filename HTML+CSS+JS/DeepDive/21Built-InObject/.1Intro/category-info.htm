표준 빌트인 객체standard built-in objects
ESMA 사양에 정의돤 객체, 애플리케이션 전역의 공통 기능. 자바스크립트 모든 실행 환경에서 동일하게 사용 가능. (이미 선언된) 전역 변수처럼 언제나 참조 가능.
호스트 객체host objects
ESMA 사양에 정의되지 않음. 실행 환경(주로 브라우저)에서 추가로 제공함.
  • 브라우저 환경: DOM, BOM, Canvas, XMLHttopRequest, fetch, requestAnimationFrame, SVG, Web Storage, Web component, Web Worker
  • Node.js 환경: Node.js 고유 API
