HTML을 장식하는 언어, CSS

CSS(Cascading Style Sheets)는 문서의 외관을 장식해준다. 전용 파일 확장자는 .css로 HTML 파일과 별개로 작성이 가능하다.

CSS를 사용하면 페이지 배경에 이미지를 설정하고, 레이아웃이나 배치를 커스터마이징하며, 문자나 그림도 장식할 수 있다.

페이지에서 CSS를 비활성화(혹은 주석 처리)하면 HTML으로 구성된 간단한 문서를 볼 수 있다. 배경 이미지나 레이아웃 등이 CSS를 적용하기 이전 상태로 돌아온다.

보기 쉬운 레이아웃 구성하기

  1. 시선의 흐름 이해하기

    사용자가 화면을 볼 때 시선 흐름 법칙이 두 가지 존재한다.
    • 1번째는, Z 법칙이다. 이미지가 많아 빠르게 읽기 좋은 사이트에 유용하다.
    • 2번째는, F 법칙이다. 정보량이 많거나 여러번 방문하는 사이트에 유용하다.

  2. 정보의 우선순위 결정하기

    페이지 안에서 어떤 요소를 가장 중요하게 보여줄 것인지 고려한다. 내용이 너무 많거나, 난잡하게 구성되지 않도록 주의한다.
    • 배치 순서는 보통 가장 먼저 읽기 시작하는 왼쪽 위를 기준으로 잡는다. 오른쪽 아래로 갈수록 우선순위가 떨어진다.
    • 배치 면적은 최상단의 메인 이미지나 주목해야 할 요소를 먼저 크게 만든다. 반면에 우선순위가 낮은 요소들은 면적을 작게 하는 것이 좋다. 물론 레이아웃의 우선순위를 먼저 고려해야 한다.
